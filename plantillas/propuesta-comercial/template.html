El desafío

{{CLIENTE_NOMBRE}} está montando un dispensario de cannabis medicinal en Chile y necesita una operación digital completa. Hoy no cuentan con una plataforma que permita a los pacientes agendar consultas médicas, ni con un sistema interno para gestionar socios, stock, entregas y contratos.

El desafío tiene dos dimensiones: una cara pública (que los pacientes puedan acceder a consultas médicas online de forma simple y segura) y una cara operativa (que el equipo interno pueda gestionar todo el flujo del dispensario desde un solo lugar).

Lo que necesitan: un ecosistema digital que conecte la experiencia del paciente (agendamiento + telemedicina) con la operación interna del dispensario (socios, stock, recetas, entregas y fidelización) — todo construido a medida y funcionando en 12 semanas.

Nuestra propuesta
Dos módulos, una operación integrada
🌐

Web de agendamiento

Landing pública con pago integrado, calendario GHL y envío automático del link de sesión médica.

⚙️

Plataforma de gestión

Webapp a medida para gestionar socios, stock, entregas con control de gramaje, contratos y fidelización.

Alcance detallado
01
Web de agendamiento + telemedicina
  • Landing pública para agendar consulta con prestador médico
  • Pago integrado con Mercado Pago — la cita se confirma al pagar
  • Calendario conectado a GHL Calendar
  • Envío automático del link de sesión (Google Meet / Zoom)
  • Notificaciones: confirmación, recordatorio y link de sesión
02
Plataforma SaaS de gestión
Gestión de socios
  • Creación por admin: nombre, RUT, dirección, teléfono, receta, gr permitidos
  • Perfil con historial de compras y saldo disponible según receta
Contratos
  • Diseño del template de contrato
  • Generación y firma digital vía GHL
Stock
  • Ingreso (bodega) → tránsito (sala de ventas) → egreso (entrega)
  • Control de inventario en tiempo real
Control de entregas
  • Validación automática contra gramaje máximo de receta
  • Bloqueo si el socio excede el límite — registro por entrega
Etiquetas + Fidelización
  • Impresión de etiquetas con datos del paciente (impresora Zebra)
  • Integración Boomerang.me vía API para gestión de puntos
Componente Herramienta
Webapp Next.js + Supabase
CRM / Agendamiento / Contratos GoHighLevel
Pagos Mercado Pago
Fidelización Boomerang.me (API)
Automatizaciones n8n
Hosting Vercel
